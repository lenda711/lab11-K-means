# Customer Segmentation Using K-Means Clustering

## Project Description
This project focuses on analyzing credit card customer behavior using the K-Means clustering algorithm. The goal is to divide customers into groups according to their purchasing patterns and financial activities.

---

## Main Steps
1. Importing and exploring the dataset  
2. Handling missing values  
3. Scaling numerical features  
4. Applying K-Means clustering  
5. Evaluating clusters using Elbow Method and Silhouette Score  
6. Visualizing customer groups with PCA

---

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
The clustering model was able to identify multiple customer segments with different spending habits. Some groups showed higher purchase activity, while others relied more on cash advances and balance usage.

---

## Objective
The purpose of this project is to understand customer behavior patterns and demonstrate the use of unsupervised machine learning techniques in data analysis.
